This page allows premium artists to create blog style announcements.
If you have a GigMan Request membership, these announcements can be displayed to your audience in your GigMan Request page.

There are three fields for each announcement:

Date:

The date the announcement should be released on. Only announcements on or prior the current date are displayed to the public.

Title

A short description of the announcement topic.

Announcement Text

The body of the announcement. Anchor tags (links) are allowed. Innapropriate links will result in user suspension.

Once an announcement has been saved, it will appear in the Existing Announcements list.

Each element in the list can be edited and resaved, or deleted.

Selecting an announcement from the list displays it in the Preview pane as it will be seen by the public in the GigMan Request Announcements window.
