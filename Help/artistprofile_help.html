From here, you can view site attributes for artists you are affiliated with.
If you are the owner of the artist account, you can also edit many of the artist attributes.

Artist Profile for

Select the current artist. Premium members can have an unlimited number of artists.

Artist name

Yep - the artists name.

Web Page

An optional web page for the artist.

Available/Selected Genres

You can specify what kinds of music your artists produces. Not currently used for much, but will be in the future.

Members

The members section displays the members and their roles registered to the selected artist profile. If you are the prime user for the artist, you can delete members by clicking the 'X' to the right in the members row.

Payments

Premium GigMan membership and GigMan Request membership can be paid for by the month or by the year. Only PayPal is currently accepted.

GigMan Request Settings

If the currently selected artist has a membership to GigMan Request, this section shows the settable properties for that account on the GigMan Request website.
GigMan Request URL Name If the currently artist has a GigMan Request account, this is the identifier used when giving your fans the GigMan Request URL.
The identifier can be changed at any time, but cannot be a duplicate of an existing identifier. The GigMan app will let you know if the identifier you have entered is a duplicate, and allow you to edit it. If you are the primary user for the artist, you can also edit these attributes.
Show Members Show the artist members in the Message target area of the GigMan Request website.
Show Categories Show/Hide the Categories column in the sample song list on the GigMan Request page.
Show Composer Show/Hide the Artist column in the public song list on artists GigMan Request page.
Enable Downvotes Allow users to downvote songs in your sample list. This can be useful for determining which songs are working for your audiences.
Max. Messages/user/24 hr Specify how many iMessages a user can send to this artist in one 24 hour period.
Max. Requests/user/24 hr Specify how many requests a user can send to this artist in one 24 hour period.

Pending Join Requests

Approve or deny requests from other GigMan users to join the currently selected artist.
