This page allows you to create calendar events, with varying levels of visibility.
If you have a GigMan Request membership, and have specified that the calendar should be shown, events marked as public will be visible to your fan-base on your GigMan Request page. There are three viewing levels:

  • Private: Only the event creator can view or edit the event.
  • Members: Only members of the creating artist can view the event.
  • Public: The event is viewable on the artists Request page, if the 'Show Calendar' option is specified.

Event Creation

Clicking any open space on a day block in the caledar will open the event editor.

Event Editing

Clicking any event title will open the event editor, with the event pre-loaded for editing. Clicking the 'x' to the left of an event will delete the event permanently.

Moving Events

An event can be dragged from one date to another.
