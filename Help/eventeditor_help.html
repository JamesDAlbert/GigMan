The event editor allows you to specify and save calendar events.

Fields

Event Date: Click the calendar icon to open a date picker for the event. This field is required.

Venue: Where the event is taking place. This serves as the event's title, and is a required field.

URL: You can add the (fully qualified) address of any website; if the event is public, clicking the title will take the user to the address specified.

Setup time, Start time, End time, # Sets: Only the start/end times are viewable by the public.

Visibility: Who can view the event. Private events are only viewable by the event creator, while public events can be viewed by the general public, if the artist has a GigMan Request account and has specified that the calendar be visible.
Members refers to members of the artist account active when the event was created.

Event Description: Details for the event.

Artist Notes: Any other details that should not be public.
