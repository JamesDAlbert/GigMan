Browsers

GigMan should run on most modern browsers, but is specifically targets Google Chrome. The reasons for this are twofold:
Google Chrome is the most used browser in the world, and Google Chrome supports many of the modern HTML features used in the site to enhance usability.
Other browsers can be used, but Google Chrome is optimal.
GigMan is written using HTML5, CSS3 and JavaScript. JavaScript must be turned on in the browser.

Interface

All of the pages in GigMan contain a header, which in turn contains some common elements for navigating the site and controlling GigMans visual presentation.

Colors There are three color schemes available:

  Hi-C This is a high contrast, black and white presentation scheme.
  Classic This is a flat styled interface using low-saturation colors.
  Neon This is a flashy, high-saturaion interface with lots of eye-candy.

The color scheme can be selected from the three position scheme switch in the header.

Navigation
   Opens the navigation panel.
   Navigates the browser to the home page.
   Navigates the browser to the previous page. This is just a more accessible 'Back' button.
   Opens the help panel for the current page or panel.

List Page Presentation
A two way switch on the right side of the header enebales switching between a standard list presentation or a tile presentation of the data being displayed.
