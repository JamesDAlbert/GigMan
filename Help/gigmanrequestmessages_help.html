Audience members can message the artist in near-real time. A message consists of an optional topic, selectable from a drop down, an optional 160 character free-text message, and an optional file attachment. Only image files are allowed as attachments.

These messages can be sent to individual artist members (if the artist and individual member profiles allow it), the leader only (the owner of the Artist Profile) or the entire band.

Messages can be viewed in the navigation popout () or the Dashboard area of the GigMan site.

The number of messages a single user can send in a 24-hour period can be set in the Artist Profile editor.
