The GigMan Request member artist can control several aspects of audience intereaction, which are settable in the Artist Profile editor, accessible in the Profiles area of the GigMan site.
The settings are

  • When an artist directs the audience to the GigMan Request site, this is the unique identifier at the end of the URL.
  • Controls whether to show individual artist members on the GigMan Request Message page. If this is turned off, only the 'Leader' and 'All' options are shown as possible recipients for messages. When it is turned on, individual members can be messaged if their personal profiles allow it.
  • Controls whether to show the 'Categories' column in the public song list. See the Song Editor help topic for an explanation of Category.
  • Controls wheter to show the Composer column in the public song list. If an artist does all origial music, it might make sense to turn this off.
  • Turning this option on allows audience members to vote songs off the island. This can be useful to the artist to determine what songs in their lineup are not working.
  • The maximum number of messages a single audience member can send in a 24 hour period.
  • The maximum number of requests a single audience member can send in a 24 hour period.
Another aspect of control is in the calendar.
Premium GigMan members can keep a calendar of upcoming events. Each event created can be set to 3 levels of privacy, one of which is 'Public'. Public events can be viewed by the public under 'Calendar' on the artists GigMan Request page.
