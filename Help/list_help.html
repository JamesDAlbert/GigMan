This page displays songs by artist, genre or category, as well as the default master song list for the current artist.

Toolbar


Hide/Show Columns: Select which columns to show.

Print List: Print all pages of the current song list.

Save List: Save the current song list as a CSV file.

Navigation Panel: Opens the navigation panel. A blinking Menu button indicates that there are Messages or Requests available.

Previous Page.: Navigate to the previous page.

H/N/C: Switch between the Hi-Contrast, Neon and Classic color schemes.

Tile/List: Switch between list and tiled display.

Filter text box: Enter text to filter songs in the list.

Columns


Edit:
Edit the song
Delete the song
Increment Play Count for the song.

Plays: Shows the current play count for the song. Hovering over the number displays the last play date, if applicable.

Title: The song title. Clicking the link takes you to the lyrics view page for the song.

Composer: The song's composer or original recorder. Clicking the link takes you to the composer's song listing page.

Genre: The song's genre. Clicking the link takes you to the genres song listing page.

Category: The song's category, if defined. Clicking the link takes you to the categorys song listing page.

Added: The date the song was added to the database.
