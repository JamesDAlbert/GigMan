This page displays lyrics and chords for a song, if they have been saved.

Toolbar


Previous Page: navigate to the last page viewed.

Navigation Panel: Opens the navigation panel. A blinking Menu button indicates that there are Messages or Requests available.

Log out Abandon your current session.

Navigate to the home page.

Song Notes: Opens the notes editor, where you can add notes about the song, and see previously saved notes.

Page Up/Down: Allows for a quick way to page up or down.

Song Properties: Opens a window showing the properties of this song.

Chords: If chords are embedded in the lyrics, and your configuration allows them to be shown, clicking a chord will display the tab for it. Clicking inside the tab window will show alternate voicings, and clicking outside the tab window will close it.

Lyric Blocks: Tapping inside a block of lyrics will collapse it so that only the first line is shown. Tapping the line will expand the block.

Left Hand Sliders: The top slider controls transposition of chords, and the bottom slider controls font size. The sliders can be shown or hidden from the Tools Panel.

Tools Panel


Transpose: If chords have been saved for the song, tapping one of the transpose buttons will transpose the chords to the key tapped. Alternately, the top left-hand scrollbar on the lyrics page can be used to transpose.

Font Size: Increases/decreases thew size of the font in the lyrics view. Alternately, the bottom left-hand scrollbar on the lyrics page can be used to change the font size.

Font Face: Changes the font used in the lyrics view. In order to ensure that chords align with lyrics, only mono-spaced fonts are available.

Display: Controls visibility of chords and left-hand sliders.
  Chords: Shows or hides the chords in the lyrics view.
  Charts: Determines if clicking a chord will show the tablature for it.
  Hints: Determines if mousing over a chord will show the tablature for it.
  Left Sliders: Show/Hide the transposition and font size sliders in the left of the window.
Colors: Allows for changing the color of the various 'parts' of a displayed song. Clicking a button opens a color chooser.

Song Notes: Opens the song notes editing window.

Increment Play Count: Updates the play count and last-played date for the current song.

Play Song Sample: If a URL has been provided for a sample of the current song, clicking this button will open the browser's default media player..
