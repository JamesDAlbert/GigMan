This panel contains links to all of the main pages on the Gigman web site.

Pages


 Master song list: A listing of all of the songs you have defined and saved.

Unplayed songs list: A listing of all songs marked as never having been played.

Composers list: A listing of all of the composers you have specified for your song data.

Genres list: A listing of all the genres specified in your song data.

Categories list: A listing of all of the categories you have specified in your song data.

Set lists: Any set lists you have defined and saved can be accessed here. Set lists are also created and edited using the editor on this page.

Editors


Song Editor: Add a song. Clicking the pencil icon in any list will also open the editor.

Site Settings Editor: Customize the appearance and behaviour of the Gigman site.

The setlist editor is available on the Setlist page.

Tools


Random Artist: Takes you to the Artist Song list page with a random artixt loaded.

Random Song: Takes you to the lyrics page with a random song loaded.

Resynch Data: If you are a subscriber, clicking this icon will ensure that your data is safely backed up in the Gigman database.

FAQ: Frequently asked questions that are not answered in these help panels.
