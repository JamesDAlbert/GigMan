Create a new artist.

Fields

Artist Name: The name of your artist.

Artist Website: Any valid webste. Websites are checked periodically for appropriateness.

Your Primary Genres: The musical genre(s) your artist identifies most with. click genres to add multiple values.

GigMan Request URL Tag: If the artist will be subscribing to the Request feature of GigMan, this value is the end of the url to which you will send your audience members. Generally, the shorter this field is, the better. GigMan will let you know if tha tag you choose is already taken.

GigMan Request Settings: If the artist will be subscribing to the Request feature of GigMan, these fields control what can be seen by the public on your Request page, and how many requests and messages a user can send in a single 24 hour period.
