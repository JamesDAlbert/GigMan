Previously created and saved setlists can be navigated to and viewed on this page.

The following navigation controls are available:

Get the previous set list (if more than one is defined).

Get the next set list (if more than one is defined).

Get the previous subset for the current setlist (if more than one is defined).

Get the next subset for the current setlist (if more than one is defined).

Note that the setlist/subset navigation wraps. For instance, if you are viewing the last subset of a setlist, and click the next subset button, then the first subset will be shown.

Open the setlist editor to create a new setlist, or modify an existing setlist.
