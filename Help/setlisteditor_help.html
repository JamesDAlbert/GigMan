Setlists can be created and edited from this toolkit. Songs from the master song list can be added to the set list on the right by clicking the image beside each song.
A setlist is comprised of three parts: The set, one or more subsets, and one or more songs in each subset.

A Set is a collection of subsets, such as the planned show for a specific venue.
A Subset is a collection of songs, such as the first set, second set etc.
Each Subset is a collection of songs. A song can appear more than once in a subset.

When searching in the song list or set list, only the currently visible columns are searched. Click the button in the list header to select the columns to display.

Create a Setlist

  • Click on the button in the setlist toolbar to clear the current list.
  • Give the set and current subset a name. Note that sets/subsets cannot be renamed. Giving a set/subset a different name, then saving it, will result in a new set/subset.
  • Add some songs from the master list to the new subset.
  • Drag the songs in the subset up or down to reorder them.
  • Remove songs from the setlist by clicking the image beside each song.
  • Click the button to save the setlist. It may take a few seconds to synchronize.

Done!
