Setlists can be created and edited from this toolkit. Songs can be dragged from the master song list onto a subset node in the treeview on the right.
A setlist is comprised of three parts: The set, one or more subsets, and one or more songs in each subset.

Show/Hide Genre/Category: If screen real estate is at a premium on your device, these two fields can be hidden to reduce the width of the song list.

Create a Setlist

  • Right click on the All Sets label, and select Create a new set from the context menu
  • Right click on the newly created 'New Set' label, and select Rename set from the menu; a text box will appear, allowing you to give the new set a name.>
  • Right click on the setlist name and select Create a new set from the context menu.
  • Right click on the newly created New subset label and give the subset a meaningful name.
  • Drag some songs from the master list onto the new subset. To drag multiple songs, click the rows to select them, then drag onto the subset.
  • Drag the songs in the subset up or down to reorder them. The songs can also be dragged into adjacent subsets.
  • Right click on the setlist name label and select Save Setlist. It may take a few seconds to synchronize.

Done!

Note that a subset must be open (click the open icon to the left of the subset name) to see all of the songs in the subset.
